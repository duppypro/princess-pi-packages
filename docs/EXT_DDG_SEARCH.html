DuckDuckGo Search (ddg-search.ts)

Domain: External Knowledge & Tool Registration

Overview

The DuckDuckGo Search extension registers a custom tool, search_web, which enables the agent to search DuckDuckGo directly from the command line using the ddgr utility. This allows the AI to fetch up-to-date documentation, API references, and modern framework guidelines.

Tool Registration

Command Execution:

The extension executes ddgr --json --noprompt --num 5 "<query>" natively using Pi's execution API, bypassing the need for an external API key or web scraping service.

Future Direction:

As per the architectural roadmap, this extension is slated for conversion into a pure Skill or Standalone Tool format rather than remaining a conventional session-hooked Extension.
