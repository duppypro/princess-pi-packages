Where The F***ing Tokens?! Utility (wtft.ts)

Domain: Cost Audit, Telemetry & Token Accounting

Overview

The WTFT (Where The F***ing Tokens?!) utility provides visual, granular accounting of your session's token consumption and associated monetary cost ($ USD). It parses the active session's append-only .jsonl log, classifies each historical interaction into one of three work types, and renders a stacked, color-coded ASCII bar graph over configurable time intervals.

Command Reference

/wtft [options]

Analyze and display the session cost breakdown in a persistent TUI widget below the editor.

Options

  • -i, --interval <size><m|h|d|w> - Group cost data into arbitrary binned intervals of minutes, hours, days, or weeks (e.g., 1m, 7m, 4h, 1d, 2w; default: 1h).
  • -l, --limit <number> - Limit the number of interval bars displayed (default: 10).
  • -w, --width <number> - Set the maximum character width of the TUI widget (default: 80).
  • -t, --tz, --timezone <zone> - Specify a target display timezone (e.g., America/New_York, Europe/London, Asia/Tokyo) to bypass UTC systems (default: system local).
  • -c, --cumulative - Render running cumulative sums (default behavior).
  • -b, --bucket - Render discrete binned interval cost buckets.
  • --ticks / --no-ticks - Enable or disable the proportional cost scale ticks above the bars (default: true).
  • -S, --show / -H, --hide - Show (force refresh) or hide the cost audit widget.

Classification Taxonomy

To avoid guessing or misrepresenting untraced exchanges, interactions are strictly bucketed into three mutually exclusive categories:

Classification Color Code Trigger Criteria
Spec Work (spec) ANSI Bright Green (\x1b[92m) Touches files under docs/ or /docs/, or matching AGENTS.md, ARCHITECTURE.md, README.md, or contains prominent spec planning keywords. Falls back to treating any .md file as spec.
Code Work (code) ANSI Orange (\x1b[38;5;208m) Touches files under .pi/extensions/, extensions/, src/, tests/, public/ (or their slash-prefixed equivalents), or executes tests/compilation terminal commands via `bash`. Falls back to treating code files (e.g., .ts, .js, .json) as code.
Other Work (other) ANSI Grey (\x1b[38;5;244m) General conversation, session initialization, model system prompts, or untraced exchanges that do not cleanly match the other two categories.

TUI Visualization Style

The TUI outputs a stacked, horizontal ASCII bar graph in a persistent widget below the editor. It uses standard block-drawing characters (█ and ░) to represent absolute cost, scaled relative to the maximum interval cost. To maximize horizontal space for bars, dates are omitted from individual time labels (which display in local hh:mm 24hr format) and instead, the most recent date is appended to the ticks labels row above the bars (if ticks are enabled). The title row displays the winged money emoji and the command name, and right-justifies the cost as a single block: Total Cost: $6.83, aligned with the rightmost boundary of the bar region. In --cumulative mode (the default), an extra field showing the incremental cost for each bucket is displayed between the time label and the cumulative cost column. If intervals cross a day boundary and --ticks is enabled, a visual day change indicator line is rendered between those intervals.

Architectural Design

1. Log Parser: Reads ctx.sessionManager.getEntries() (which includes raw historical messages, tool calls, and exact usage/costs) or reads the `.jsonl` session file directly if more raw history is needed.

2. Binning Engine: Groups entries by their timestamp into chronological bins based on the requested interval (e.g., 6-minute windows for granular auditing, or 1-day windows for multi-day tracks).

3. Scaling & Layout: Normalizes the absolute length of each interval's ASCII bar based on the maximum cost in any single interval (typically capped at 30-40 characters wide to fit standard terminal outputs gracefully). Each sub-segment (Spec, Code, Other) within that bar is allocated its proportional share of characters based on its absolute cost contribution, without rendering percentages or additional trailing text.
